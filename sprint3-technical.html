    Describe what the DOM is and an example of how you might interact with it.

DOM = Document Object Model

The DOM is an application that orders the contruction of a website, it allows the user to access various aspects of the webpage. This allows them to manipulate and change the page as they will. This changes the local version of the webpage. It is designed to be used with any programming language.

    Explain the difference between accessing data from arrays and objects.

Array = is a set/group of variables, you can use an array to group these objects under a single name, this name can be used to manipulate all the variables inside the array.

Object = Objects are named keywords, representing given values in the script. objects are everything excluding javascript primitives - eg. string, numbers, booleans, null and undefined. Hoever if some of these are defined with keywords they can becaome objects. Eg. a number or set of numbers can be named to become a variable and then they are now objects.
